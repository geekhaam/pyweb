※ 코드와 데이터베이스를 연결하는 API를 사용해보자.

- 코드 종류별로 여러가지가 있다.
- 우리가 사용할 것은 PHP MYSQL API
- 내장된 명령어들을 찾아서 사용해보자.
	1) 접속
	   : mysql_connect
	2) 작업 영역 설정 (DATABASE)
	   : mysql_select_db()
	3) 어떤 작업(작업 범위)을 할 것인지 결정 (CRUD)
	   : mysql_query()
	4) 접속 종료
	   : mysql_close()

* php.net에서 mysql api 검색

<작업 순서>

 1. php 코드로 만든 파일이 필요하다. (htdocs)
 2. 이 파일 안에서 데이터베이스에 접속 -> 데이터베이스 선택
    -> 작업 -> 접속 종료, 이와 같은 코드가 들어간다.
 3. 기존에 cmd에서 하던 작업을 그대로 코드로 옮긴다고 보면 된다.
 4. 파이썬도 마찬가지이지만, 드라이버 종류와 명령어 이름 등이 다르다.
 5. 이런 작업을 로우 레벨 api 작업이라고 한다.
